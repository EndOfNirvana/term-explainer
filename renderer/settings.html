术语解释器设置

配置大模型 API，选中术语后用快捷键一键解释

API 配置
只需填基础路径，程序自动拼接 /chat/completions
如 gpt-4o-mini / deepseek-chat / qwen-turbo / glm-4-flash / MiniMax-M3
快捷键
选中词语后按此键，弹出三种解释。默认 Ctrl+Shift+D
选中英文后按此键，弹出中文翻译 + 三种解释。默认 Ctrl+Shift+T
默认 Escape。常用：Escape / F4 / Alt+F4
使用方式
🔍 解释模式（默认 Ctrl+Shift+D）
   选中任意术语 → 按快捷键 → 学术定义 + 两种通俗解释

🌐 翻译模式（默认 Ctrl+Shift+T）
   选中英文词语/句子 → 按快捷键 → 中文翻译 + 学术定义 + 两种通俗解释

点击弹窗中任意卡片可复制内容
