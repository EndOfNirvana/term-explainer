# 更新日志

## v3.3.0

- 新增学术模式（独立快捷键 `Ctrl+Shift+S`），蓝色主题弹窗
- LLM 解释 + Semantic Scholar + OpenAlex 三路并行请求
- 底部展示两篇真实文献，标注来源数据库名称，点击打开论文页
- 设置页新增学术检索快捷键配置，托盘菜单新增学术检索入口
- 完善 README：全功能表格、三种模式对比、API 详解、架构说明

## v3.2.0

- 新增英文翻译模式（独立快捷键 `Ctrl+Shift+T`）
- 翻译弹窗绿色主题，含中文翻译卡片
- 两种模式快捷键均可在设置页自定义

## v3.1.0

- 移除硬编码 `temperature` / `max_tokens`（兼容 Kimi 等模型）
- 移除默认模型 `gpt-4o-mini`，用户必须自行填写
- 推送到 GitHub，配置 Actions 自动打包 Win + Mac

## v3.0.0

- API 调用从 `node-fetch` 改为 Node.js 原生 `https` 模块（零外部依赖）
- 新增四级网络诊断（DNS / TCP / TLS / HTTP）
- 新增 VPN / 代理环境自动检测
- 设置页新增「诊断网络」按钮
- 设置页窗口在聚焦时不再响应全局快捷键
- 弹窗关闭键可自定义（`cancelKey` 配置项）

## v2.1.0

- 自定义快捷键（查询键、关闭键）
- 多弹窗可拖动、并存
- Win32 `keybd_event` 读取选中文本
- API 调用移至主进程 IPC，含 5 层 JSON 降级解析
- 设置页快捷键不误触发

## v1.0.0

- 初始版本
- 划词查询 + 三种解释
- OpenAI 兼容接口
- 系统托盘常驻
